# Data
Data collected during INSPACe CANSAT 2025, please refer to the following table for the units and details -

| **Parameter**                | **Description**                            | **Unit** |
| ---------------------------- | ------------------------------------------ | -------- |
| **Time**                     | Timestamp of the measurement               | s        |
| **Packet**                   | Telemetry packet count                     | —        |
| **State**                    | System or flight state indicator           | —        |
| **Altitude**                 | Measured altitude (from barometric sensor) | m        |
| **Pressure**                 | Atmospheric pressure                       | hPa      |
| **Temperature (Barometric)** | Ambient temperature (from BMP sensor)      | °C       |
| **Temperature (DHT)**        | Ambient temperature (from DHT sensor)      | °C       |
| **Humidity**                 | Relative humidity                          | %        |
| **Voltage**                  | System/battery voltage                     | V        |
| **GPS Latitude**             | Latitude coordinate                        | °        |
| **GPS Longitude**            | Longitude coordinate                       | °        |
| **GPS Altitude**             | Altitude from GPS module                   | m        |
| **Acceleration (X, Y, Z)**   | Linear acceleration components             | m/s²     |
| **G-Force**                  | Resultant acceleration magnitude           | g        |
| **Gyro Rate**                | Angular rate (from gyroscope)              | °/s      |
| **PM2.5**                    | Particulate matter concentration           | µg/m³    |
| **CO (Coal Gas)**            | Carbon monoxide concentration              | ppm      |
